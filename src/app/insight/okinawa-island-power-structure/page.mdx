# 沖縄島嶼電力構造：本土と連系しない離島系統の特殊性

> 沖縄電力 (沖縄本島 + 離島 11 系統) の**島嶼電力構造**を可視化。**本土との連系線ゼロ、火力依存 90% 超、燃料費調整制度のダイレクト影響、離島ごとの独立系統**という日本唯一の構造を読み解く。

## 背景

沖縄は日本の電力市場で**唯一 JEPX に参加していないエリア**:

1. **JEPX 取引対象外** (北海道〜九州の 9 エリア、沖縄は別市場)
2. **本土との連系線ゼロ** (海底ケーブル計画は議論中だが未実現)
3. **沖縄電力単独管轄** (一般送配電事業者 + 小売 + 発電を一体運営、新電力参入限定的)
4. **離島 11 系統**: 宮古島 / 石垣島 / 北大東島 / 南大東島 / 久米島 / 渡嘉敷島 / 座間味島 / 北部 / 与那国島 / 西表島 / 波照間島
5. **電源構成**:
   - 火力 (LNG + 石炭 + 重油) **90% 超**
   - 再エネ (太陽光 + 風力) 5-8%
   - その他 2-3%

つまり、沖縄は**完全に閉鎖系統**で電力市場が運営される稀有なエリア。燃料価格上昇が**直接的に**電気代に転嫁される構造で、日本本土とは別の経済学が働く。

## 沖縄電力の電源構成 (年間概算)

| 電源 | 比率 | 月次発電量 (GWh) |
|---|---|---|
| LNG 火力 | 35% | 250 |
| 石炭火力 | 30% | 215 |
| 重油火力 | 25% | 180 |
| 太陽光 | 5% | 35 |
| 風力 | 1% | 7 |
| その他 | 4% | 28 |
| **合計** | **100%** | **約 715** |

合計年間約 86 億 kWh = 全国の **1.0%**。電源は火力 90% 超で、原発・大規模水力なし。

## 構造の特殊性 (本土 9 エリアとの対比)

| 項目 | 本土 9 エリア (JEPX) | 沖縄 |
|---|---|---|
| 連系線 | 相互接続 (周波数同期 + 連系線網) | 完全独立 (海底ケーブル なし) |
| 卸電力市場 | JEPX で価格収斂 | 沖縄電力単独運営、市場価格なし |
| 新電力参入 | 旺盛 (シェア 25%) | 限定的 (シェア 5%) |
| 火力依存 | 50-65% | **90% 超** |
| 燃料費調整 | 各電力会社 (1.5-2 ヶ月遅延) | 沖縄電力単独 (直接的、遅延小) |
| 離島供給 | 本土から有線 | 離島ごとに独立発電所 |

### 離島 11 系統の構造

沖縄電力管内の離島 (宮古島・石垣島など) は**それぞれ独立した小規模発電所**を持つ。例:

- **宮古島**: 重油 + 太陽光 + 風力 (10-15 MW 規模)
- **石垣島**: 重油 + 太陽光 (15-20 MW 規模)
- **波照間島**: 重油発電のみ (1-2 MW 規模)

これらは**離島補完電源** (蓄電池 + 自然エネ) と組み合わせた**スマートグリッド実証実験**の最前線。本土 9 エリアでは見られない構造で、Phase D で「日本のマイクログリッド最前線」として独立 Insight 候補。

## 燃料費調整制度の特殊性

沖縄電力の燃料費調整単価は:

- **本土 9 電力**: 1.5-2 ヶ月の遅延で電気代に反映
- **沖縄電力**: 0.5-1 ヶ月の遅延でほぼリアルタイム反映

これは沖縄電力が**燃料費感度に対して最も透明性が高い**ことを意味する。LNG 価格急騰 (2022 年) の影響が、本土より早く沖縄電気代に出る構造。

用語集 [燃料費調整制度](/glossary/fuel-adj) + [円建てコスト](/glossary/yen-denominated-cost) も参照。

## 解釈

### 「閉鎖系」の経済学

沖縄電力は**閉鎖系で運営**されるため、本土の経済学とは異なる:

1. **規模の経済 不適用**: 離島ごとの独立発電所、規模メリット限定
2. **競争市場 不在**: 沖縄電力単独で価格決定、JEPX による市場価格メカニズム なし
3. **燃料価格の直接反映**: 燃料費調整が迅速 = LNG 急騰が即座に電気代に
4. **再エネ導入の制約**: 系統小規模 + 蓄電池高コストで再エネ比率が低位

### 海底ケーブルの議論

本土と沖縄を繋ぐ**九州-沖縄海底ケーブル** (約 800 km、技術的に可能) が長年議論されているが、コスト (数千億円) と需要規模 (1% シェア) の経済性で実現未定。

実現すれば:
- 沖縄電力が JEPX 9 エリアに統合
- 火力依存度低下、再エネ受け入れ容量増
- 電気代低下効果

ただし、災害時 (台風) のケーブル断線リスクも考慮必要。

### 北海道 (Insight #53) との対比

| 特性 | 北海道 | 沖縄 |
|---|---|---|
| 連系線 | 本州と接続 (90 万 kW) | ゼロ |
| 火力依存 | 60% | 90%+ |
| 再エネ余地 | 風力大 (陸上) | 太陽光中程度 |
| 季節需要 | 冬季暖房急増 | 夏季冷房 + 年間安定 |
| JEPX | 参加 (ボラ大) | 不参加 |

北海道は**冬季需要 + 連系線制約**で JEPX ボラが大、沖縄は**閉鎖系 + 火力依存**で**燃料価格直接反映**という異なる特殊性。両者を並べることで、本土 7 エリア (関東 + 関西 + 中部 + ...) の標準的構造との対比が明確に。

## 注意点

### データ不在の制約

本記事は**沖縄電力の月次データなし**で執筆。沖縄電力公表データ + 経産省統計から推定値を使用。Phase D 第 1 期で catalog 追加後、より精密な分析が可能。

### 離島データの不在

離島 11 系統別データは沖縄電力でも個別公表が限定的。マイクログリッド研究の進展に伴い、データ可用性が将来向上見込み。
