# 日本のエネルギーは誰から買っているか：原油・LNG・石炭の輸入相手国

> 2025年、日本は原油・LNG・石炭を計約18.7兆円輸入した。原油は中東に94%依存、LNGは豪40%を中心に分散、石炭は豪67%の一極集中。同じ「化石燃料の輸入」でも、燃料ごとに供給リスクの形がまるで違う。

## 背景：エネルギー安全保障は「どこから買うか」

日本は一次エネルギーの自給率が低く、化石燃料の大半を輸入に頼る。だからエネルギー安全保障の核心は「どの燃料を・**どこの国から**買っているか」にある。相手国が偏れば、その国・地域の地政学リスク（供給途絶、海上輸送路の封鎖、価格交渉力の低下）がそのまま日本のエネルギー安定に跳ね返る。

EIC Data の地政ドメインに、**財務省貿易統計（e-Stat）の相手国別輸入額**が着地した。2025年について、原油・LNG・石炭の輸入を相手国別に並べてみる。

## 読み解き

1. **原油＝中東一極（94%）**：UAE とサウジの 2 国だけで 83%。日本の原油はほぼ中東から来ており、ホルムズ海峡という単一の海上輸送路に強く依存する。供給途絶リスクが最も高く、価格交渉力も働きにくい構造だ。
2. **LNG＝多様化が進む**：最大の豪州でも 40% で、マレーシア・カタール・米国・パプアニューギニア・ブルネイに分散している。**米国は 7%**——シェールガス由来の米 LNG は近年に登場した新しい供給源だ（`jp-import-value-lng-us` は 2017 年以降に立ち上がる）。一方で**ロシアが 9%** 残っており、「脱ロシア」が一筋縄でいかない現実も映る。
3. **石炭＝豪依存（67%）**：次点のインドネシア（15%）を大きく引き離し、豪州一国に偏る。原油の中東依存とは別の意味で、特定国への集中が際立つ。

要するに、同じ「輸入化石燃料」でも、**原油は中東一択・LNG は多様化・石炭は豪集中**と、リスクの形が燃料ごとに異なる。エネルギー安全保障を一括りで語れない理由がここにある。

> 注: 本記事は過去データ（2025年）の記述的分析であり、将来予測ではない。輸入「額」は 価格 × 数量 × 為替 の合成であって、数量ベースの依存度とは一致しない（円安や価格高騰の年は金額が膨らむ）。相手国別の金額は財務省貿易統計の HS 品目別データに基づく。
