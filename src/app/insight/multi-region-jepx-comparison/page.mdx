# 9 エリア JEPX 比較：地域差の構造と原因

> JEPX 9 エリア（北海道 + 東北 + 東京 + 中部 + 北陸 + 関西 + 中国 + 四国 + 九州）の日次価格を 1 ページで比較。電源構成の違い（原発 / 太陽光 / 火力 / 水力）が地域別の価格水準と振幅にどう反映されるかを可視化。Insight #31-#34 のヒートマップシリーズの **導入記事**。

## 背景

日本の電力市場は **9 つの地域エリア**に分かれており、エリアごとに JEPX 価格が形成される。連系線（北本 + 東地域 + 中部関西 + 関西中国 + 中国九州 + 四国本州）で電力が融通されるが、容量制約があり、エリア間で価格差が生じる。

地域別の価格水準と振幅は、**電源構成の違い**を反映:

- **関西** (原発 7 基) + **九州** (原発 4 基): 価格本州最安
- **中部** (浜岡停止) + **東京** (柏崎刈羽停止): 火力依存で本州最高
- **北陸** (水力ベース) + **東北** (火力 + 風力): 中位
- **北海道** (火力 + 風力 + 連系線制約): 寒波時は最高
- **四国** (伊方 1 基依存): 振幅が大きい

## 解釈

**3 つのグループ**:
1. **低価格安定型**（関西 + 九州 + 北陸）: 原発再稼働 + 水力 + 太陽光大量導入で安定
2. **中位中振幅型**（東京 + 中部 + 中国 + 東北）: 火力中心で燃料連動強
3. **高振幅型**（北海道 + 四国）: 連系線制約 + 単一原発依存で局面的に高騰

**連系線の意義**: 関西 → 中国 → 九州、東京 → 中部 → 関西 で電力が融通されるが、容量制約で完全均一化されない。連系線増強（北本 90→150 万 kW など）が進めば価格差が縮小する見込み。

**ヒートマップシリーズへの予告**: 本記事で見た「9 エリア × 月次価格」の構造は、Insight #31-#32 の「**9 地点 × 月次気象**」と並べて読むと、**価格と気象の地理的対応**が見えてくる。

**価格水準と振幅の独立性**: 関西と九州は同じ平均価格（9 ¥/kWh）だが、九州はさらに小振幅（太陽光大量導入の昼間反転効果）。原発再稼働がベースロードを支え、太陽光が日中ピークを抑える、**原発 + 太陽光の組合せ効果**。

## 注意点

- JEPX エリア（電力会社管内）と 9 地点気象（気象庁管区）は完全に一致しない（北陸が 3 県、四国が 4 県など）
- 卸価格と小売価格は別、家庭電気料金は燃料費調整制度経由で遅延反映
