# 「EIC Data」一般公開のお知らせ — 日本のエネルギーと金融の引用インフラ

公開日: 2026 年 5 月 12 日 (火)
発信元: 一般社団法人エネルギー情報センター 代表理事 江田健次

---

## 概要

一般社団法人エネルギー情報センター (代表理事 江田健次) は、エネルギー・金融・マクロ経済データの引用可能な公共基盤「EIC Data」(<https://data.eic-jp.org/>) を 2026 年 5 月 12 日 (火) より一般公開しました。当初計画 (6 月 29 日 GA) から 48 日の前倒しでの公開となります。

「EIC Data」は、研究者・ジャーナリスト・実務者が「2 クリックで一次出典に辿れる」基準点として活用できる、無料・無広告・収益化なしのデータ基盤です。当法人姉妹サイト「新電力ネット」(<https://pps-net.org/>) に続く第 3 プロダクトとして公開します。

---

## 「EIC Data」とは

「日本のエネルギーと金融の交差点」に焦点を絞った、引用可能な公共データ基盤です。米国の FRED (連邦準備銀行) はマクロ経済データ、英国の Our World in Data はグローバル社会課題を中心としており、いずれも日本独自の電力卸価格・気象・燃料市場を一次出典付きで横断するサービスは存在しませんでした。「EIC Data」はこの空白を埋めるため、JEPX (日本卸電力取引所) の電力卸価格 10 エリア・JMA (気象庁) 観測値 9 地点・燃料市場・金融指標 (USD/JPY・JGB・米マクロ) を横断的に解析記事化し、各系列に source_url・license・observation_cutoff・as-of を一意付与しています。

全 105 系列を国際標準ライセンス **CC BY 4.0** で提供し、データ販売や有料プランの設定は行いません。引用元として、学術論文・報道記事・行政資料での使用を想定した設計です。

---

## 主要機能 5 つ

**1. 独自 Insight 41 本** — 6 軸 (マクロ 9 + 気象×電力 10 + 燃料 6 + 電源構成 7 + 気候×地理 5 + 需給 4) で日本の電力・燃料市場の構造を読み解く長尺記事を公開しています。

**2. catalog 105 系列** — 4 ドメイン (電力 22 + 気象 63 + 燃料 7 + 金融 13) の時系列データを、CSV 一括ダウンロード (約 2.25 MB) で取得可能です。各系列に 19 項目のメタデータ (一次出典 URL・ライセンス・観測値最終確定日等) を D-011 スキーマで明示しています。

**3. 朝刊サマリー `/today`** — 毎平日朝 7:00 JST に 5 系列を横断する自動更新解説を配信。過去 5 日分のアーカイブ閲覧が可能で、編集者バッジ + トレンドアラート (異常変動の検出) を併記しています。

**4. Indicator API + 一括ダウンロード** — CORS 対応の REST API で 105 系列を外部 BI ツール・研究スクリプトから直接読み込み可能。CSV 一括 DL は約 2.25 MB と軽量で、無料・無認証で利用できます。

**5. 引用フォーマット 3 形式** — 各 Insight および catalog 系列に CitationButton を常設し、BibTeX (LaTeX 論文用) ／ Chicago 17th ／ APA 7th をワンクリックで生成。学術論文や記事執筆での出典記述を効率化します。

## 開発の経緯と特徴

「EIC Data」は、Anthropic 製の Cowork mode + Claude Code (CLI 型コーディングエージェント) を活用した **8 日間集中スプリント** (2026-05-05 β公開 → 2026-05-12 GA) で構築されました。AI 編集長ペルソナ「リン」と Claude Code 実装エージェントの協業により、当初プラン (6 月 29 日 GA) から 48 日前倒しの公開を実現しています。

特に 5 月 11 日 (月) 9:00-20:00 の 11 時間で 9 連続マイルストーンを達成、翌 5 月 12 日 (火) に機能完成日達成 + 統合テスト全 3 日完走 (全 207 URL HTTP 200 OK / Lighthouse 全 12 主要ページ Performance 平均 0.972 / Core Web Vitals 完全クリア) + 大規模告知準備 + 法務最終確認 + GA 達成という 7 連続を集約し、2 日間で 16 連続マイルストーンを樹立しました。

品質面では、Lighthouse Performance / Accessibility / Best Practices / SEO の 4 カテゴリすべてで 0.93 以上 (全 12 主要ページ平均: Performance 0.972 ／ Accessibility 0.99 ／ Best Practices 1.00 ／ SEO 1.00 ／ CLS 0.004) を達成。モバイル・タブレット・デスクトップの 3 ビューポート × 主要 12 ページのレイアウト崩れ検査も 0 件で、ブラウザ互換性 (Chrome 86+ / Edge 86+ / Safari 15.4+ / Firefox 85+) も確認済です。
